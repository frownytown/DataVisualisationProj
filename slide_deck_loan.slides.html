Investigation Overview¶

For this Data Exploration I will be taking a look at the Prosper Loan Data set, which contains data about 113,937 loans with 81 variables for each of them. This includes things like loan amount, interest rate, current loan status, borrower income, etc.

Dataset Overview¶

This dataset is structered in 81 columns by 113937 rows. This dataset is all variable related to Loans.

Loan Status¶

Here we can see the relative outcomes for all the loans. As we can clearly see, the vast majority of outcomes seem to be successful, with "Current" and "Completed" taking up the lions share. This does seem to suggest that the Prosper company has some way to determine whether or not a loan will be successful.

Correlation of BorrowerAPR and its strongest relationships¶

We can see from this bar plot of BorrowerAPR's strongest relationships the various features that are most related to it. We can see the obvious ones like it being correlated with itself, as well as being strongly correlated with BorrowerRate (Which makes sense since the BorrowerRate is part of how the APR is calculated), but we also see some strange things like LenderYield and EstimatedReturn being quite high. Paradoxically, EstimatedLoss is also quite strongly correlated. We do see that the Prosper Rating is strongly negatively correlated so that if someone has a low Prosper Rating they will have a high APR, this makes sense as well since if someone has a lower Prosper Rating the Prosper system probably sees them as a risky loan option, thus they offset it with a high APR.
